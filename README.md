# WEDU 
FabLab **W**orkshop **EDU**cation

WEDU is a collaboration between the Kattegatt gymnasium and Halmstad University to
educate Students in the FabLab Workshop.

This is a collection of all the master-files for the assignments. 
Assignments are designed to teach one or more machines and can come in different levels of difficulty to complete. 
Designs and drawings are made to reflect real industrial parts.
